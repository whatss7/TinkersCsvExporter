# Tinkers' CSV Exporter

## Overview

A simple client-side mod for exporting material data from [Tinkers' Construct 3](https://github.com/SlimeKnights/TinkersConstruct/). Requires Tinkers' Construct 3 (if not obvious).

For Minecraft 1.16.5 - 1.19.2, simply join the world and CSV files will be generated in the `tcexporter` folder. This behavior is disabled if any CSV file already exists in the folder.
For Minecraft 1.20.1, run `/tcexporter export` to generate files containing all material data in the `tcexporter` folder.
